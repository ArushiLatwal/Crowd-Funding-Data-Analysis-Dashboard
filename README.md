Kickstarter Crowd Funding Data Analytics Project
This repository showcases an end-to-end data analytics project on Kickstarter crowdfunding data. The project analyzes campaign performance, funding success rate, project categories, and backer engagement. The workflow includes data analysis using SQL and interactive dashboards built using Excel, Power BI, and Tableau to generate meaningful insights about crowdfunding trends and project success factors.

01_Excel_Dashboard  :    	Excel dashboard analyzing project outcomes, categories, and funding trends
02_PowerBI_Dashboard:	    Power BI dashboard showing campaign success rate, backers, and amount raised
03_Tableau_Dashboard:	    Tableau dashboard providing interactive visualizations of crowdfunding performance
04_SQL_Scripts	    :     SQL queries used for KPI calculations and data analysis
Data	              :     Raw dataset containing Kickstarter crowdfunding campaign data
visualizations_preview:	  Dashboard screenshots for quick preview
README.md            :    Main project documentation

Data Source
The dataset contains information about Kickstarter crowdfunding campaigns (2009–2019) including:
Project Name
Category
Country / Location
Goal Amount
Amount Raised
Number of Backers
Project Outcome (Successful, Failed, Cancelled, etc.)
Launch Date
Campaign Duration

Dashboards
Excel Dashboard
Provides insights into:

Total number of projects by outcome
Top projects by backers
Amount raised by top campaigns
Projects distribution by category
Project trends by year, quarter, and month
PowerBI Dashboard
Analyzes crowdfunding performance with KPIs such as:
% of successful projects
Total number of backers
Average number of days for successful projects
Total amount raised by successful campaigns
Project distribution by category and country

Tableau Dashboard
Interactive visualizations including:
Total projects and categories
Percentage of successful projects
Amount raised by successful campaigns
Number of projects by country and category
Success rate by goal range

SQL Analysis
SQL scripts were used to perform the following analysis:
Calculate project success rate
Analyze funding by category and country
Identify top successful campaigns
Evaluate goal amount vs raised amount
Analyze project trends over time

Visualizations Preview

Screenshots of Excel, Power BI, and Tableau dashboards are available in the visualizations_preview folder to quickly view the insights without opening the files.

How to Use

Review the dataset in the Data folder.

Run SQL queries from 04_SQL_Scripts to perform analysis.

Explore dashboards in Excel, Power BI, and Tableau folders.

Check dashboard screenshots in visualizations_preview for quick insights.
